# Security Policy

## Reporting

Do not open public issues for security findings. Instead:

- Email: security@phenotype.space (or kooshapari@gmail.com until org mailbox exists)
- Disclosure: Please allow 48 hours for initial response

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Out of Scope

- Social engineering attacks
- Physical security
- Denial of service attacks against public infrastructure
